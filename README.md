# Login-test
This is a simple project with a login microservice (back-end) and a web page (front-end)


## Back-end
The back-end was developed using SpringBoot with the starter web and starter data components, the project usaes a h2 database to store the data model


## Front-end
The front-end is simple web page that uses jQuery to make Ajax calls to the back-end services exposed.


## Testing the Solution
For install an test the solution please review the [docs](https://github.com/ecarrilloq/login-test/tree/master/docs) folder on this repository
