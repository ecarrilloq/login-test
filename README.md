# Login-test
This is a simple project with a login microservice (back-end) and a web client (front-end) that consumes the login service exposed by the back-end microservice


## Back-end
The back-end is developed using SpringBoot with starter web and starter data, the project usaes a h2 database to store the data model


## Front-end
The front-end is simple page that uses jQuery to make Ajax calls to the back-end services exposed.


## Testing the Solution
For install an test the solution please review the [docs](https://github.com/ecarrilloq/login-test/tree/master/docs) folder on this repository
